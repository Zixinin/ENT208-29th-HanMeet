# HanMeet
A Chinese learning APP for global exchange students

## Project Description
An APP that provides AI-powered Chinese pronunciation feedback and global text-based language partner support, helping international students prepare for Chinese proficiency exams.

## Core Functions
1. AI Chinese Pronunciation Correction & Feedback
2. Global Language Partner Text Help & Practice
